# E-Shop - Electronic Store Project

## Project Overview
An e-commerce website for small/medium businesses specializing in handmade jewelry and eco-friendly products.

## Features
- Product catalog with 20+ products in 3 categories
- Product reviews and ratings
- Modern responsive design
- Shopping cart functionality
- Category filtering

## Categories
1. **Handmade Jewelry** - Unique handcrafted jewelry pieces
2. **Eco-Friendly Food** - Organic and sustainable food products
3. **Home Decor** - Handmade home decoration items

## Technology Stack
- HTML5
- CSS3 (Responsive Design)
- JavaScript (ES6+)
- Local Storage for cart and reviews

## Setup
Simply open `index.html` in a web browser to view the store.
